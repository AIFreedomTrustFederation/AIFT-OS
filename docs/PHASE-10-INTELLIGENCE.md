# Phase 10: Real Federation Intelligence

The intelligence layer is based on real repository state only.

It inspects:

- actual Git status
- actual files present
- actual manifests
- actual capability records
- actual detected frameworks
- actual generated reports

It does not simulate maturity. It reports confidence, score, role, recommendations, and orchestration readiness from evidence.

Commands:

- `aift intelligence scan`
- `aift intelligence report`
- `aift intelligence repo <repo>`
- `aift intelligence roadmap`

Generated files:

- `registry/intelligence.json`
- `reports/intelligence.md`
