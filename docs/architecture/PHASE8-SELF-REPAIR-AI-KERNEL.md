# AIFT-OS Phase 8: Self-Repair AI Kernel

The Self-Repair AI Kernel is conservative.

It inspects repository reality, repairs safe generated state, creates safe scaffolds, runs validation, and reports blocked issues.

It must not fake capabilities, invent active systems, delete source code, or overwrite human work.

Commands:

- aift ai
- aift ai doctor
- aift ai repair --safe
- aift ai verify
- aift ai report
