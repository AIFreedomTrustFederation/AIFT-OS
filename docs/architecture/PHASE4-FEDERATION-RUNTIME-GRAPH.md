# AIFT-OS Phase 4: Federation Runtime Graph

AIFT-OS builds a runtime graph from discovered reality.

The graph is not hardcoded.

The graph is built from repository evidence on disk.

## Discovery Targets

- Git repositories
- Package manifests
- Node projects
- Go projects
- Rust projects
- Python projects
- Docker files
- Compose files
- GitHub Actions workflows
- OpenAPI specs
- GraphQL schemas
- MCP indicators
- Ollama indicators
- AI provider indicators
- Documentation
- Services
- Commands
- Dependencies

## Rule

Repositories are source packages.

AIFT-OS owns the runtime graph.
